# 刘语悦 - 周岁宴电子邀请函 🍼

这是一个基于 HTML5 + CSS3 + 原生 JavaScript 开发的移动端单页邀请函（H5），专门为刘语悦小朋友的一周岁生日宴打造。以萌趣粉色调、奇幻童话故事书（Storybook）为主题风格，采用温暖的渐变配色、手绘风格动画和童趣字体，深度适配手机端显示。

## 🌟 核心功能

1. **响应式与视觉还原**：专为移动端（手机浏览器、微信内置浏览器）设计的自适应布局，深度还原邀请函参考图的粉嫩色调与童话感排版。
2. **渐入动画与视觉特效**：页面拥有动态漂浮的星星、爱心云朵背景，各个模块在向下滚动时平滑淡入（Fade-in）。
3. **成长图鉴首屏展示**：打开页面的第一眼即是“成长图鉴”滑动相册，使用了 `Swiper.js` 实现拍立得卡片堆叠滑动浏览体验，每张照片配备专属语录。
4. **定制欢乐背景音乐**：网页右上角包含常驻的跳动音乐播放按钮（带有移动端触屏自启交互优化），配合活泼可爱的纯音乐。
5. **一键导航系统**：宴会信息区集成了高德地图“一键导航”能力，点击即刻唤起手机端高德APP进行导航。
6. **飞书多维表格集成 (RSVP)**：
   - 底部回执表单能够让来宾填写姓名、赴宴人数及留言防忌口。
   - 所有数据将通过飞书开放平台直接 push 至指定的“飞书多维表格”。

## 🛠️ 如何使用和部署

### 本地预览
**极简方式**：只需在您的电脑或手机上直接使用浏览器（推荐 Chrome 或 Safari，手机微信直接打开最佳）打开 `index.html` 文件即可。
> **测试建议**：在电脑浏览器按 `F12` 开启开发者工具，切换至移动设备视角（例如 iPhone 14 Pro）以获得最佳阅览效果。

### 替换内容

- **替换音乐**：若需更换现有音乐，准备新的首选 mp3 文件，重命名为 `bgm.mp3` 覆盖原文件。
- **替换照片与配套文案**：将裁剪好的照片放入 `/photos` 目录。打开 `main.js`，找到顶部的 `photos` 数组，修改对应的路径和文字说明即可自动呈现在首屏图鉴中。
- **修改邀请文案**：直接使用任意代码编辑器或记事本打开 `index.html`，找到“麻麻说，我要请客啦”对应的文字段落修改。

### 🚀 关于飞书接口与生产环境发布

邀请函底部的 RSVP 表单配置了您的飞书凭证。请注意：

由于浏览器严格的**跨域资源共享 (CORS) 安全策略**限制，前端页面如果是直接部署在公网域名下（例如 GitHub Pages），直接向 `https://open.feishu.cn` 发送 POST 请求会被浏览器拦截并报错。

#### 解决方案：
1. **轻量级分享（不部属于公网）**
   如果您只是把这些文件打包成压缩包发给亲友，或者利用如局域网共享打开本地文件，通常不会受到严格 CORS 限制。
2. **通过 Serverless / 云函数做转发（推荐用于大规模分享）**
   最稳妥的做法是配合 `uniCloud` 或 Cloudflare Workers 写一个云端转发接口，由前端调用不受 CORS 限制的服务器环境发起真实请求。
